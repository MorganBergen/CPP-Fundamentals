Post-Lab - Lab (8) - Overloading Operators - Rational Class


In the post-lab assignment for Lab(6), post-lab, you wrote a program that used a Rational Number Class to perform some computation.  Write a new program or modify that program so that it uses operator overloading for all the operations performed in that assignment (+, -, *, and /).

In addition, overload operators:
    1)  == ,
    2) <= ,
    3) >= ,
    4) > , and
    5) <.
