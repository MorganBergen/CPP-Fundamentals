Post-Lab - Lab (13) - Reverse Order an Integer


1) Write a C++ program that displays a number's digits in reverse order.  Thus, the program will take a number such as 345 and will display the result as 543.

You are required to give two options to the user:
        Option 1) Use a recursive process to find the number in reverse order
        Option 2) Use an iterative process to find the number in reverse order
